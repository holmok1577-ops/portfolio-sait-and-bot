# Компетенции: промпт-инженеринг

## Что умеет промпт-инженер

Промпт-инженер формулирует инструкции для языковой модели так, чтобы ответы были точными, стабильными и полезными для бизнес-задачи.

## Ключевые навыки

- Понимание сильных сторон и ограничений LLM.
- Разработка system prompt, user prompt и шаблонов сообщений.
- Настройка роли, тона, границ ответственности и формата ответа.
- Использование zero-shot, few-shot и пошаговых подходов.
- Тестирование промптов на типовых и пограничных вопросах.
- Улучшение ответов через итерации и анализ ошибок.
- Проектирование диалоговых сценариев и пользовательского пути.

## Для чего это нужно в проекте

Хороший системный промпт помогает ассистенту отвечать в нужном стиле, ссылаться на источники, не выдумывать факты и корректно эскалировать сложные случаи человеку.

## Пример правил ассистента

- Отвечай кратко и по делу.
- Используй базу знаний как главный источник.
- Если ответа нет в документах, скажи честно.
- Для коммерческих запросов предложи связаться со специалистом.
