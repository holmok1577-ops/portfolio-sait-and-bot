# Проект: Portfolio Site & AI Assistant

## Ссылка

GitHub: https://github.com/holmok1577-ops/portfolio-sait-and-bot

## Назначение

Проект представляет собой сайт-портфолио Станислава с интегрированным AI-ассистентом. Ассистент помогает посетителям узнать о компетенциях, услугах, проектах, процессе работы и возможностях AI-решений.

## Основные функции

- Чат-виджет на сайте.
- Переключение режимов: RAG и обычный AI-помощник.
- Подключенная база знаний в Markdown/TXT.
- Telegram-бот с тем же ассистентом.
- Админ-панель для логов, статистики, изображений и документов.
- Форма обратной связи с уведомлениями.
- Docker и скрипты деплоя.

## Технологии

- Python, FastAPI, Uvicorn.
- ChromaDB для векторного поиска.
- OpenAI API через ProxyAPI.
- SQLite и SQLAlchemy.
- Vanilla HTML/CSS/JavaScript.
- Docker, Docker Compose, Nginx, systemd.

## Ценность проекта

Проект демонстрирует полный цикл разработки AI-ассистента: постановку задачи, подготовку данных, RAG, интерфейс, Telegram-интеграцию, мониторинг, деплой и документацию.
